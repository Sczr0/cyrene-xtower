## 项目多语言支持现状

Cyrene 项目目前支持以下语言：

- 英语 (en) - 作为基础语言
- 简体中文 (zh-cn) - 默认语言
- 繁体中文 (zh-tw) - 使用简体中文作为回退

## 技术栈

- **Inlang Paraglide-js** - 多语言框架
- **SvelteKit** - 应用框架
- **TypeScript** - 类型系统
- **自定义本地化系统** - 结合 Inlang 消息格式和自定义类型定义
